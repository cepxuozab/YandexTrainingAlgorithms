## A. Все перестановки заданной длины

По данному числу N (0 < N < 10) выведите все перестановки чисел от 1 до N в лексикографическом порядке.  

Пример 1
### Ввод
```cpp
1
```

### Вывод  
```cpp
1
```

Пример 2
### Ввод
```cpp
2
```

### Вывод  
```cpp
12
21
```

Пример 3
### Ввод
```cpp
3
```

### Вывод  
```cpp
12
2123
132
213
231
312
321
```
