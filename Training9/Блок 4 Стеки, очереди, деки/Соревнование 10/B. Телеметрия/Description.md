## 🧩 Задача: Телеметрия текстовых окон

Во время соревнования по программированию используется специальный браузер, который собирает телеметрию.

На компьютере участника открыто $n$ текстовых окон, в каждом отображается текстовый файл. На момент начала работы все
окна пустые. В каждом окне в любой момент отображаются последние $k$ символов текстового документа.

Есть несколько кнопок, которые может нажать участник:

* **Латинские строчные буквы**
  В конец текущего текстового файла добавляется символ, соответствующий нажатой кнопке.

* **Backspace**
  Из конца текущего текстового документа удаляется один символ. Если документ пустой, ничего не происходит.

* **Copy**
  Та часть текстового файла, которая видна на экране, сохраняется в буфер обмена.

* **Paste**
  В конец текущего текстового файла добавляется то, что лежит в буфере обмена. Изначально буфер пуст.

* **Next**
  Открывает следующее окно. Порядок обхода окон всегда одинаковый. После последнего окна кнопка открывает первое.

---

## 📥 Формат ввода

В первой строке вводится три натуральных числа $n$, $m$, $k$
$(1 \le n \le 10;\ 1 \le m, k \le 1000)$ — количество окон, количество нажатий клавиш и ограничение на количество
отображаемых символов.

В следующих $m$ строках вводятся описания нажатых клавиш — строчные латинские буквы, либо название клавиши из описанных
выше.

---

## 📤 Формат вывода

Выведите строку, которая будет отображаться на экране после нажатия всех клавиш.

Если текущий текстовый файл пустой, выведите:

```
Empty
```

---

## 📌 Пример 1

**Ввод:**

```
2 7 10
Next
a
b
Copy
Next
Paste
Paste
```

**Вывод:**

```
abab
```

---

## 📌 Пример 2

**Ввод:**

```
10 7 10
a
b
Copy
Backspace
Paste
Paste
Backspace
```

**Вывод:**

```
aaba
```
