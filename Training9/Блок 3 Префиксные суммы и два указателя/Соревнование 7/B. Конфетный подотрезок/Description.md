# B. Конфетный подотрезок

* ограничение по времени на тест 2,5 секунды
* ограничение по памяти на тест 256 мегабайт

В ряд выложено $n$ конфет. У каждой конфеты есть тип $p_i$. Можно выбрать последовательность подряд идущих конфет с
одним условием — в этой последовательности должно быть ровно два различных типа конфет.

Определите максимальное количество конфет, которое можно взять.

### Формат ввода

В первой строке вводится одно натуральное число $n$ ($n \le 10^6$) — количество конфет.

Во второй строке вводится $n$ целых чисел $p_i$ ($1 \le p_i \le 10^9$), где $p_i$ — тип $i$-й конфеты.

### Формат вывода

Выведите максимальное количество конфет, которое можно взять.

### Пример 1

#### Ввод

```
6
3 3 1 2 2 1
```

#### Вывод

```
4
```

### Пример 2

#### Ввод

```
2
1 1
```

#### Вывод

```
0
```
