## Сумма на отрезке без изменений

Дан целочисленный массив $A$, состоящий из $N$ элементов.

Обработайте $Q$ запросов следующего вида:

$L$ $R$ — определить сумму элементов с индексами от $L$ до $R$ включительно.

Так как количество запросов в этой задаче велико, их аргументы не будут описаны во входных данных. Вместо этого вам
нужно получить их при помощи генератора случайных чисел.

Вам нужно сгенерировать последовательность чисел $X_0, X_1, \ldots, X_{2Q-1}$. Число $X_0$ указано во входных данных, а
остальные числа вычисляются по формуле
$$X_i = (11173 \cdot X_{i-1} + 1) \bmod 1000000007.$$

Далее из этих чисел образуются аргументы запросов:

- $L_0 = \min(X_0 \bmod N, X_1 \bmod N)$, $R_0 = \max(X_0 \bmod N, X_1 \bmod N)$;
- $L_1 = \min(X_2 \bmod N, X_3 \bmod N)$, $R_1 = \max(X_2 \bmod N, X_3 \bmod N)$;
- ...
- $L_{Q-1} = \min(X_{2Q-2} \bmod N, X_{2Q-1} \bmod N)$, $R_{Q-1} = \max(X_{2Q-2} \bmod N, X_{2Q-1} \bmod N)$.

Для большей ясности прочитайте примечание к примеру.

### Формат ввода

Первая строка содержит целое число $N$ ($1 \le N \le 10^5$) — размер массива.

Вторая строка содержит $N$ целых чисел $A_i$ ($0 \le A_i \le 10^9$) — элементы массива.

Третья строка содержит целое число $Q$ ($1 \le Q \le 10000$) — количество запросов.

Четвёртая строка содержит целое число $X_0$ ($0 \le X_0 \le 10^9$) — начальное значение генератора случайных чисел.

### Формат вывода

Выведите одно целое число — остаток от деления суммы ответов на все запросы на $1000000007$.

## Примеры

**Ввод**

```
5
1 2 3 4 5
3
123
```

**Вывод**

```
27
```

**Ввод**

```
7
34 10 49 32 28 15 70
5
0
```

**Вывод**

```
492
```  

### Примечание

Последовательность $X$ в первом примере равна $\{123, 1374280, 354830336, 519316381, 321884300, 413258729\}$.

Запросы:

- $L_0 = 0$, $R_0 = 3$ (ответ $10$);
- $L_1 = 1$, $R_1 = 1$ (ответ $2$);
- $L_2 = 0$, $R_2 = 4$ (ответ $15$).

Сумма ответов на все запросы равна $27$.
