## Префиксные суммы

По данной последовательности $a_1, a_2, \ldots, a_n$ вычислите последовательность ее префиксных
сумм $b_1, b_2, \ldots, b_n$, где $b_j = \sum_{i=1}^{j} a_i$.

### Формат ввода

В первой строке дано целое число $n$ ($1 \le n \le 10^3$) — количество элементов в последовательности $a$.  
Во второй строке дано $n$ целых чисел $a_1, a_2, \ldots, a_n$ ($|a_i| \le 10^6$) — элементы последовательности.

### Формат вывода

Выведите $n$ целых чисел $b_1, b_2, \ldots, b_n$ — последовательность префиксных сумм для последовательности $a$.

## Примеры

**Ввод**

```
5
10 -4 5 0 2
```

**Вывод**

```
10 6 11 11 13 
```
