# 1. Закраска клеток

* ограничение по времени на тест 1 секунда
* ограничение по памяти на тест 256 мегабайт

Робот находится на бесконечном клетчатом поле. Программа для робота состоит из команд $U$ (сдвинуться на клетку
вверх), $R$ (сдвинуться на клетку вправо), $D$ (сдвинуться на клетку вниз) и $L$ (сдвинуться на клетку влево). Робот
закрашивает все клетки, на которых он побывал, включая начальную.

Определите, сколько клеток будут закрашены более одного раза.

### Формат ввода

В единственной строке содержится строка $S$ с программой для робота. Она состоит только из символов $U$, $R$, $D$ и $L$,
и её длина не превосходит $1000$:
$$|S| \le 1000.$$

### Формат вывода

Выведите количество клеток, закрашенных более одного раза.

### Пример 1

#### Ввод

```
ULDR
```

#### Вывод

```
1
```

### Пример 2

#### Ввод

```
ULRD
```

#### Вывод

```
2
```
