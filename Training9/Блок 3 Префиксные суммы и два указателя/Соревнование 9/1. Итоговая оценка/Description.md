# 1. Итоговая оценка

* ограничение по времени на тест 1 секунда
* ограничение по памяти на тест 256 мегабайт

В течение семестра каждый студент получал различные оценки, записываемые латинскими буквами от "A" до "Z". При этом "
A" — это лучшая оценка, а "Z" — худшая. При подсчете итоговой оценки преподавательница считала среднее арифметическое
всех оценок, полученных за семестр, и округляла его по обычным правилам округления (в сторону билжайшего целого). В
случае, если оценка одинаково далеко от двух ближайших целых — она округлялась в сторону лучшей оценки. Однако, по ее
предмету нельзя было получить оценку более чем на балл превосходящую худшую оценку в течение семестра.

Посчитайте итоговую оценку по известным оценкам за семестр.

### Формат ввода

В единственной строке содержится непустая строка, состоящая из заглавных латинских букв, длина которой не превышает 100.

### Формат вывода

Выведите один символ — итоговую оценку за семестр.

### Пример 1

#### Ввод

```
ABACABA
```

#### Вывод

```
B
```

### Пример 2

#### Ввод

```
AZAA
```

#### Вывод

```
Y
```

### Пример 3

#### Ввод

```
ABABAB
```

#### Вывод

```
A
```
