# A. Большое и кратное трем

**ограничение по времени на тест:** 1 секунда  
**ограничение по памяти на тест:** 256 мегабайт

Нужно из набора цифр составить наибольшее возможное число, которое будет делиться на три без остатка. При этом, число
может начинаться с ведущих нулей, и при равных значениях большим считается более длинное. Например, "00021" считается
большим, чем "021".

### Формат ввода

Вводится строка, состоящая из цифр от 0 до 9 — набор цифр. Длина строки не меньше трех и не превосходит $10^5$.

### Формат вывода

Выведите наибольшее число, которое можно составить из данных цифр, чтобы оно делилось на три без остатка. Гарантируется,
что это возможно.

### Примеры

**Ввод**

```
105
```

**Вывод**

```
510
```

**Ввод**

```
2222
```

**Вывод**

```
22
```  

**Ввод**

```
000
```

**Вывод**

```
000
```  

**Ввод**

```
54321
```

**Вывод**

```
54321
```
