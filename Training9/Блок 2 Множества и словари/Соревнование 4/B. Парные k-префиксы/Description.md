# B. Парные k-префиксы

**ограничение по времени на тест:** 2 секунды  
**ограничение по памяти на тест:** 256 мегабайт

Есть n слов одинаковой длины. Необходимо найти такое наибольшее число k, чтобы можно было разбить слова на пары так,
чтобы у каждой пары слов совпадало хотя бы k первых букв.

Определите максимальное значение k.

## Входные данные

В первой строке вводится целое число n — количество слов ($1 \le n \le 2 \cdot 10^5$, n - четное).
В следующих n строках заданы слова. Гарантируется, что все слова имеют одинаковую длину и суммарная длина слов не
превышает $2 \cdot 10^6$.

## Выходные данные

В единственной строке выведите число k — искомое максимальное значение.

### Примеры

**Ввод**

```
4
bbbb
aabc
aacc
bbbd
```

**Вывод**

```
2
```

**Ввод**

```
2
a
b
```

**Вывод**

```
0
```
