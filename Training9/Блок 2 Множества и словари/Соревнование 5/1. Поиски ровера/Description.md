# 1. Поиски ровера

**ограничение по времени на тест:** 1 секунда  
**ограничение по памяти на тест:** 256 мегабайт

Ровер-доставщик выехал из точки $0$ в начале проспекта (проспект — это прямая).
В какой-то момент спутниковая навигация отключилась и точное положение ровера
(точка $X$) оказалось неизвестным.

По сигналу с $n$ базовых станций мобильной сети удалось определить $n$ пар чисел:
$x_i$, $d_i$, означающих, что базовая станция в точке $x_i$ находится от положения
ровера $X$ на расстоянии не более чем $d_i$.

Определите наибольшее возможное значение $X$.

## Формат ввода

В первой строке вводится одно натуральное число $n$ ($1 \leq n \leq 10^5$).

В следующих $n$ строках вводятся по два целых числа $x_i$, $d_i$
($0 \leq x_i \leq 10^9$, $0 \leq d_i \leq 10^9$).

## Формат вывода

В единственной строке выведите наибольшее $X$, удовлетворяющее всем показаниям
сигналов с базовых станций, либо $-1$, если не существует $X$,
удовлетворяющего всем показаниям.

### Примеры

**Ввод**

```
2
1 2
3 2
```

**Вывод**

```
3
```

**Ввод**

```
3
2 2
6 2
8 1
```

**Вывод**

```
-1
```  

**Ввод**

```
5
100 97
115 104
97 115
111 115
107 97
```

**Вывод**

```
197
```
