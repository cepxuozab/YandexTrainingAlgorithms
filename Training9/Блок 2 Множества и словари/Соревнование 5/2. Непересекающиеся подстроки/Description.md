# 2. Непересекающиеся подстроки

**ограничение по времени на тест:** 1 секунда  
**ограничение по памяти на тест:** 256 мегабайт

Дана строка $s$, состоящая из строчных латинских символов. Необходимо найти в ней
подстроку, у которой существует максимальное количество непересекающихся вхождений
в строку $s$.

Вхождением подстроки $t$ в строку $s$ будем называть пару индексов $(l, r)$, такую,
что $l \leq r$ и подстрока строки $s$ с $l$ по $r$ символ включительно равна строке $t$.

Непересекающимися будем называть два вхождения $(l_1, r_1)$ и $(l_2, r_2)$, такие,
что отрезок $[l_1, r_1]$ не пересекается с отрезком $[l_2, r_2]$.

Найдите максимальную длину подстроки, удовлетворяющей условию.

## Формат ввода

Дана строка $s$ ($1 \leq |s| \leq 10^5$), состоящая только из строчных латинских букв.

## Формат вывода

Выведите максимальную длину подстроки, удовлетворяющей условию.

### Примеры

**Ввод**

```
abacaba
```

**Вывод**

```
1
```

**Ввод**

```
abab
```

**Вывод**

```
2
```  

### Примечание

В первом примере такой строкой является строка "a".

Во втором примере такой строкой является строка "ab".
