# Экзамен в бункере

Вася работает преподавателем в университете, и ему надоело, что студенты пользуются LLM для решения задач, поэтому он
взял в аренду для проведения экзамена бункер с металлоискателем на входе. Теперь он хочет красиво рассадить студентов и
сделать селфи со студентами на фоне.

Всего у Васи **n** студентов. Их необходимо рассадить по следующим правилам:

* Студенты должны быть рассажены по рядам.
* В любых двух рядах число студентов должно различаться не более чем на один.
* Если есть ряды с различным числом студентов, то в любых двух соседних рядах число студентов должно быть различным.

Вася не хочет, чтобы помещение для рассадки студентов было сильно вытянуто по вертикали или горизонтали, так что его
интересует минимальный модуль разности числа рядов и максимального числа студентов в одном ряду.

---

## Формат ввода

В единственной строке содержится натуральное число **n** — число студентов:($1 \le n \le 10^{12}$)

## Формат вывода

Выведите одно число — минимальный возможный модуль разности числа рядов и максимального числа студентов в одном ряду.

---

## Примеры

### Пример 1

**Ввод**

```
1
```

**Вывод**

```
0
```

---

### Пример 2

**Ввод**

```
2
```

**Вывод**

```
1
```

---

### Пример 3

**Ввод**

```
3
```

**Вывод**

```
0
```

---

### Пример 4

**Ввод**

```
50
```

**Вывод**

```
3
```
