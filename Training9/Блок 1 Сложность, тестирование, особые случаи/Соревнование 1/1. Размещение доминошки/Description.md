# Размещение доминошки

Дано клетчатое поле размером n × m. Некоторые клетки поля заняты. Определите, сколькими способами можно положить
доминошку размером 2 × 1 (её можно также повернуть на 90 градусов), так, чтобы ни одна клетка доминошки не располагалась
в занятой клетке.

### Формат ввода

Первая строка содержит два целых числа n и m ($1 ≤ n, m ≤ 1000$) — размеры таблицы.

Каждая из следующих n строк содержит по m символов. Символ "#" означает, что соответствующая клетка занята, а
символ "." — что она свободна.

### Формат вывода

Выведите число способов размещения доминошки.

## Пример 1

## Ввод

```text
2 3
...
...
```

## Вывод

```text
7
```

## Пример 2

## Ввод

```text
3 4
..#.
#..#
..#.
```

## Вывод

```text
5
```

## Пример 3

## Ввод

```text
5 8
........
.##..##.
........
#......#
.######.
```

## Вывод

```text
34
```
