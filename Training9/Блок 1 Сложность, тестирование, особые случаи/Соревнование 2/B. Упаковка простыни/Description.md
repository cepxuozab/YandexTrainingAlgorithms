# B. Упаковка простыни

Длины сторон простыни равны `n` и `m` сантиметрам. Ее необходимо упаковать в плоский пакет прямоугольной формы, длины
сторон которого равны `h` и `w` сантиметрам.

Простыню можно складывать пополам по горизонтали или вертикали, это можно сделать несколько раз. Кроме того, простыню
можно повернуть на 90 градусов. Стороны сложенной простыни должны быть параллельны сторонам пакета, а размеры сложенной
простыни в соответствующих измерениях не должны превышать размеры пакета.

Определите минимальное число складываний простыни, необходимое для того, чтобы она помещалась в пакет.

## Формат ввода

В единственной строке вводятся четыре натуральных числа `n`, `m`, `h` и `w` — длины сторон простыни и пакета
соответственно ($1 ≤ n, m, h, w ≤ 10^{18}$).

## Формат вывода

Выведите минимальное необходимое количество складываний.

## Пример 1

**Ввод:**

```
10 20 20 10
```

**Вывод:**

```
0
```

## Пример 2

**Ввод:**

```
3 3 2 2
```

**Вывод:**

```
2
```
